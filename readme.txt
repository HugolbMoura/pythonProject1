Instruções de Execução para o Programa de Carteira/Corretora de Dados

Pré-requisitos:
- Python 3.6 ou superior deve estar instalado no seu sistema.
- A biblioteca tkinter deve estar disponível em sua instalação Python. (Ela geralmente vem pré-instalada com o Python.)

Instruções:
1. Salve todos os arquivos do programa (incluindo o main.py e quaisquer outros módulos necessários) em uma pasta no seu computador.

2. Abra o terminal ou prompt de comando.

3. Navegue até o diretório onde você salvou os arquivos usando o comando 'cd'. Por exemplo:
   cd Caminho/para/sua/pasta

4. Execute o programa utilizando o comando 'python' seguido do nome do arquivo principal. Por exemplo:
   python main.py
   Se você estiver usando um sistema Unix (Linux/Mac), talvez precise usar 'python3' em vez de 'python':
   python3 main.py

5. A interface gráfica do usuário (GUI) deve abrir em uma nova janela. Siga as instruções na tela para utilizar o programa.

6. Para adicionar um novo cliente ou fornecedor, utilize os botões 'Adicionar Cliente' ou 'Adicionar Fornecedor', preencha os campos solicitados e clique em 'Enviar'.

7. Para editar ou remover um cliente ou fornecedor, primeiro selecione o nome desejado na lista e então use os botões 'Editar' ou 'Apagar'.

8. Feche a aplicação normalmente através da interface gráfica quando terminar de utilizar o programa.

Notas Adicionais:
- Se você encontrar algum problema ao executar o programa, verifique se todos os pré-requisitos estão atendidos e que você está utilizando a versão correta do Python.
- Para suporte adicional, consulte a documentação do Python ou tkinter, ou entre em contato com o desenvolvedor do programa.
